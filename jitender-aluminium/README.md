# JITENDER Aluminium & Glass Work Website

A modern, responsive website for JITENDER Aluminium & Glass Work built with Next.js, Tailwind CSS, and GSAP animations.

## Project Overview

This website showcases the services offered by JITENDER Aluminium & Glass Work, a premium aluminium and glass solutions provider based in Rohini, Delhi. The site features a clean, professional design with vibrant accents and smooth animations to engage visitors and highlight the company's expertise.

## Features

- **Responsive Design**: Fully responsive layout that works on all device sizes
- **Modern UI**: Clean, professional design with the company's color scheme (Red, Yellow, Blue)
- **Animations**: Smooth GSAP animations for enhanced user experience
- **Service Showcase**: Detailed pages highlighting all services offered
- **Gallery**: Portfolio section with lightbox functionality
- **Contact Form**: Easy-to-use contact form with validation
- **SEO Optimized**: Proper meta tags and semantic HTML structure

## Technology Stack

- **Frontend Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: GSAP (GreenSock Animation Platform)
- **Icons**: React Icons
- **Deployment**: Vercel-ready structure

## Pages

1. **Home** (`/`) - Hero section, Why Choose Us, Featured Services
2. **Services** (`/services`) - Detailed service offerings including Aluminium Doors, Window Partitions, etc.
3. **Gallery** (`/gallery`) - Portfolio of completed projects with lightbox view
4. **Contact** (`/contact`) - Contact form and business information

## Color Scheme

- **Primary Red**: #E53935 (Used for CTAs, active links, key highlights)
- **Secondary Yellow**: #FFC107 (Used for contrast elements, secondary buttons)
- **Primary Blue**: #1976D2 (Used for important text, headers)
- **Background/Text**: White/Light Gray

## Getting Started

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Deployment

The website is ready for deployment on Vercel. Simply connect your GitHub repository to Vercel for automatic deployments.
